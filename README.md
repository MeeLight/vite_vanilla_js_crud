## 🎯 Setup

### 📦 Installation

```shell
pnpm i
```

### 🧪 Run Dev

```shell
pnpm dev
```

### 🚀 Run Preview

```shell
pnpm preview
```

### 🔨📦 Sass Build

```shell
pnpm sass:build
```

### ⚡📦 Vite Build

```shell
pnpm build
```
